SWE API Endpoints

  • GET /api/swe/data
    Get snow water equivalent data
  • GET /api/swe/predictions
    Get SWE predictions and forecasts
  • GET /api/swe/historical
    Access historical SWE data

Flood Warning API

  • GET /api/v1/flood/real-time-risk
    Get current flood risk assessment
  • GET /api/manitoba-flood/status
    Manitoba-specific flood monitoring
  • GET /api/manitoba-flood/alerts
    Regional flood alerts and warnings
