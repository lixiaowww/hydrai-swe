System Overview

HydrAI-SWE is an advanced hydrological intelligence system designed for snow water equivalent (SWE) prediction, flood warning, and agricultural decision support. The system integrates machine learning models with real-time environmental data to provide accurate predictions for water resource management.

SWE Prediction

Advanced machine learning models for predicting snow water equivalent with confidence intervals and seasonal analysis.

Flood Warning

Real-time flood risk assessment with early warning alerts and risk level classification.

Agricultural Intelligence

Soil moisture prediction, crop recommendations, and yield forecasting for agricultural decision support.

Data Analytics

Historical trend analysis, seasonal patterns, and comprehensive data visualization tools.

System Architecture: The HydrAI-SWE system is built with a FastAPI backend, real-time data integration from Environment Canada (ECCC), USGS, and NASA SMAP, and a responsive web frontend supporting English and French languages.

Getting Started

Quick Start

  1. Access the Dashboard

    Navigate to the main dashboard at /ui to view the primary interface with historical analysis and forecasting tools.

  2. Select Your Language

    Choose between English and French using the language switcher in the top navigation bar.

  3. Explore the Data

    Use the date range selectors and region filters to explore historical SWE data and trends for your area of interest.

  4. Generate Predictions

    Switch to the "Future Forecast" tab to access 7, 14, and 30-day SWE and runoff predictions with confidence intervals.

First Time Users: Start with the dashboard's preset date ranges (Last Year, 3 Years, 5 Years) to familiarize yourself with the data visualization tools before exploring custom date ranges.

Agriculture Applications

The Agriculture Intelligence Suite provides specialized tools for agricultural decision support, accessible through the /applications interface.

Soil Moisture Prediction

Input Parameters:

  • Location: Specify geographic region (e.g., Manitoba Province)
  • Date Range: Select start and end dates for prediction period
  • Soil Type: Choose from available soil classifications

Output Metrics:

  • Mean soil moisture percentage
  • Moisture range (min-max values)
  • Model confidence information
  • LSTM model architecture details
Model Training: The soil moisture predictor uses an LSTM neural network that may require training on first use. Training typically takes 30-120 seconds depending on data volume.

Crop Recommendations

Environmental Inputs:

  • Average temperature (°C)
  • Expected precipitation (mm)
  • Soil moisture level (%)
  • Soil type classification

Recommendation Categories:

  • Highly Suitable: Crops with optimal growing conditions
  • Suitable: Crops with good growing potential
  • Moderately Suitable: Crops that may require additional management

Yield Prediction

Predict crop yields based on:

  • Crop type selection
  • Planting date
  • Geographic location
  • Weather condition inputs

Prediction Output:

  • Expected yield value with units
  • Confidence interval range
  • Uncertainty quantification
  • Model performance metrics

Data Authenticity Report

Access detailed analysis of data sources and quality metrics through /applications/data-authenticity:

  • Data source verification
  • Quality assessment scores
  • Authenticity confidence levels
  • Recommendations for production use

Data Sources

HydrAI-SWE integrates data from multiple authoritative sources to ensure accuracy and reliability:

Primary Data Sources

Environment Canada (ECCC)

Real-time meteorological and hydrological data from Environment and Climate Change Canada monitoring networks.

USGS Water Services

United States Geological Survey hydrometric data for cross-border watershed monitoring.

NASA SMAP

Soil Moisture Active Passive satellite data for large-scale soil moisture monitoring.

Provincial Networks

Provincial water survey networks and automated snow pillow monitoring stations.

Data Quality Assurance

  • Real-time Validation: Automatic quality checks on incoming data streams
  • Historical Consistency: Long-term data consistency verification
  • Multi-source Verification: Cross-validation between different data sources
  • Gap Filling: Intelligent interpolation for missing data points (limited to 7-day gaps)

Data Update Frequency

  • Real-time Data: Updated every 15-30 minutes
  • Forecast Models: Updated every 6 hours
  • Historical Analysis: Updated daily
  • Satellite Data: Updated every 2-3 days

Troubleshooting

Common Issues

Charts Not Loading

  • Check Internet Connection: Ensure stable internet connectivity
  • Clear Browser Cache: Clear cache and refresh the page
  • Try Different Date Range: Some periods may have limited data
  • Check Region Selection: Ensure selected region has available data

No Data Available Message

  • Expand Date Window: Try a longer time period
  • Select Different Region: Some regions may have sparse data
  • Check Station Status: Verify monitoring stations are active

Agriculture Model Training Timeout

  • Wait for Completion: Initial training may take up to 2 minutes
  • Check Model Status: Use the model status endpoint to verify readiness
  • Retry After Training: Subsequent predictions will be faster

API Request Failures

  • Check API Endpoints: Verify correct endpoint URLs
  • Validate Parameters: Ensure all required parameters are provided
  • Check Network Connectivity: Verify server accessibility
Debug Mode: Open browser developer tools (F12) to view console messages for detailed error information. Look for network errors or JavaScript exceptions.

API Usage

HydrAI-SWE provides a comprehensive REST API for programmatic access to all system functionality.

Base URL

https://your-domain.com/api

Authentication

Currently, the API uses IP-based access control. Contact your administrator for API access credentials if required.

Common Endpoints

SWE Data Endpoints

# Get historical SWE data GET /api/swe/historical?start_date=2024-01-01&end_date=2024-08-20®ion=all # Get SWE forecast GET /api/swe/forecast?days=30&forecast_type=swe®ion=all # Get seasonal analysis POST /api/swe/analysis { "mode": "seasonal", "column": "snow_water_equivalent_mm" }

Agriculture Endpoints

# Soil moisture prediction POST /api/v1/agriculture/soil-moisture/predict { "location": "manitoba-province", "start_date": "2024-01-01", "end_date": "2024-08-20" } # Crop recommendations POST /api/v1/agriculture/crop/recommend { "location": "manitoba-province", "temperature": 20, "precipitation": 400, "soil_moisture": 30, "soil_type": "clay" }

Flood Warning Endpoints

# Flood risk assessment GET /api/v1/flood/risk-assessment?days=14®ion=red-river-basin # Real-time flood risk GET /api/v1/flood/real-time-risk

Response Format

All API responses follow a consistent JSON format:

{ "status": "success", "data": { // Response data }, "metadata": { "timestamp": "2024-08-20T14:30:00Z", "version": "1.0.0" } }
API Documentation: Complete API documentation with interactive examples is available at /docs (FastAPI Swagger interface).
